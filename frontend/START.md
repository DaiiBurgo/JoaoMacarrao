# 🚀 Guia Rápido para Iniciar

## ⚡ 3 Passos Simples

### 1️⃣ Instalar Dependências
```bash
cd frontend
npm install
```
**Tempo estimado:** 2-3 minutos

### 2️⃣ Iniciar Servidor
```bash
npm run dev
```
**Tempo estimado:** 10-20 segundos

### 3️⃣ Abrir no Navegador
```
http://localhost:3000
```
