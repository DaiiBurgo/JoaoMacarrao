📌 OPÇÃO 3: USAR CMD MANUALMENTE
════════════════════════════════════════════════════════════

   1. Pressione Win + R
   2. Digite: cmd
   3. Pressione Enter
   4. Cole este comando:
   
      cd C:\Users\Nilson\João Macarrão\frontend
   
   5. Pressione Enter
   6. Cole este comando:
   
      npm run dev
   
   7. Pressione Enter
   8. Abra: http://localhost:3000


════════════════════════════════════════════════════════════

📌 OPÇÃO 4: RENOMEAR PASTA (SOLUÇÃO PERMANENTE)
════════════════════════════════════════════════════════════

   1. Renomeie a pasta:
      De: C:\Users\Nilson\João Macarrão
      Para: C:\Users\Nilson\JoaoMacarrao
   
   2. Abra PowerShell nessa nova pasta
   
   3. Rode:
      npm install
      npm run dev
